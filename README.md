# solOSis
my very first webOS, made with HTML, CSS and JavaScript.
to drag any window, click on the window's header on top. press the X on the edge of the header to close the window.
if you close the welcome window, you can re-open it by clicking the image on the top-bar on the left.
to open any desktop app, you must click it once to select it, and once again to launch the window.
the AboutMe app is a static page with just an image and text.
the Paint app is a re-use of the code i posted on my PixelArt-Canvas repository.
the System app tracks the user's system data and displays it on the webOS.
the Notes app is just two text inputs.
the Calculator is a css grid of buttons with a display that tracks the numbers and operations in a string and solves it when the result button is clicked.
the Weather app uses Open Map Weather API to work.
